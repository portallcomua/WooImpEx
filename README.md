# WooImpex — імпорт товарів для WooCommerce

Простий плагін для імпорту товарів з CSV файлів.

## Можливості

- Імпорт назви, ціни, опису, артикулу, кількості
- Категорії (через `/`) та теги (через `,`)
- Основне фото та галерея зображень
- Автоматичне зіставлення колонок
- Оновлення існуючих товарів за SKU

## Як користуватись

1. **Підготуйте CSV файл** (приклади в папці плагіна)
2. **Завантажте** через сторінку WooCommerce → WooImpex Імпорт
3. **Зіставте колонки** (плагін підбере автоматично)
4. **Натисніть імпорт**
