# WooImpex Pro — імпорт товарів для WooCommerce

Простий та зручний плагін для імпорту товарів з CSV файлів у WooCommerce.

## Можливості

- 🇺🇦 **Українські заголовки** — працюйте зі зрозумілими назвами колонок
- 🎯 **Автоматичне зіставлення** — плагін сам підбирає відповідності між CSV і полями товару
- 📸 **Зображення та галерея** — підтримка основних фото та фотогалереї
- 🔄 **Оновлення товарів** — оновлюйте існуючі товари за артикулом (SKU)
- 📊 **Попередній перегляд** — перевіряйте дані перед імпортом
- 🔌 **Проста інтеграція** — працює відразу після активації

## Встановлення

1. Скачайте ZIP-архів з [Releases](https://github.com/portallcomua/wooimpex/releases)
2. Зайдіть в WordPress: **Плагіни → Додати новий → Завантажити**
3. Виберіть скачаний ZIP-файл та натисніть "Встановити"
4. Активуйте плагін
5. Перейдіть в **WooCommerce → WooImpex Імпорт**
