# WooImpEx# WooImpex – простий імпорт/експорт товарів для WooCommerce

Легкий, зрозумілий плагін, створений за філософією JBZoo: мінімум кліків, автоматичне зіставлення полів, без зайвих налаштувань.

## Можливості

- **Імпорт товарів з CSV** у два кроки:
  1. Завантажити файл (роздільник – кома, UTF-8)
  2. Перевірити автоматичне зіставлення колонок або вибрати вручну
- **Експорт товарів** в CSV однією кнопкою (всі поля: назва, опис, SKU, ціни, склад, атрибути, категорії, мітки)
- **Автоматичне створення глобальних атрибутів** при імпорті
- **Оновлення існуючих товарів** за SKU
- **Завантаження прикладу CSV** з усіма можливими полями
- **Ніяких складних налаштувань** – тільки вибір файлу та мапінг

## Вимоги

- WordPress 5.6+
- WooCommerce 6.0+
- PHP 7.4+

## Встановлення

### З ZIP-архіву (рекомендовано)

1. Завантажте останню версію з [Releases](https://github.com/portallcomua/Wooimpex/releases)
2. В адмінці WordPress перейдіть **Плагіни → Додати → Завантажити**
3. Виберіть ZIP-файл та натисніть **Встановити зараз**
4. Активуйте плагін

Після активації в меню **Товари** з'явиться підпункт **WooImpex імпорт товарів**.
